About

This front end is made to be used in cooperation with a TypeScript project running in Vite. That is to say, the controls are a hybrid, some things adjusted here and others in the source code.

This program remembers your settings. When you change the code, Vite will restart the program, and the program picks up exactly where you left off. If you're not watching closely you might not even realize that the program restarted.
