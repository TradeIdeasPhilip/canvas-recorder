# Saved State

This folder contains a copy of the changes that the user makes in the Visual Editor.
These changes are stored in IndexedDB, but that is not visible to git.

Save your files here with the button at the top of the application.

Each video gets its own section of IndexedDB and each one gets its own file in this directory.

Status: As of 6/19/2025 this is a very early prototype.
It is not yet read by the software, but it will be.
See ["Closing the Loop"](../../development-plans/visual-editor.md#closing-the-loop) for more details.
